# Release notes 0.9.0

This release finalises the first public review workflow.

- Completing a task in Workspace immediately refreshes every open Markdown view of its source note.
- Reading mode and source mode stay aligned with Workspace mutations.
- Editing a task from Workspace uses the same synchronisation path.
- Clicking a task opens Edit task; clicking its Markdown icon opens the source note.
- Review creation, duplicate handling and Review-only workspace classification remain unchanged.
