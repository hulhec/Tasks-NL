# Contributing

Thanks for helping improve Tasks NL.

## Reporting bugs

Please create a GitHub Issue.

Include

- Obsidian version
- Tasks NL version
- Steps to reproduce
- Expected behaviour
- Actual behaviour

## Pull Requests

Please discuss major features before starting implementation.

Small improvements and bug fixes are always welcome.

## Coding guidelines

- Markdown remains the source of truth.
- Keep classes focused.
- Keep UI simple.
- Follow existing formatting.

Thank you!
