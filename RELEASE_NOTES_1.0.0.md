# Tasks NL 1.0.0

First stable release of Tasks NL.

## Highlights
- Natural Dutch task capture with standard Markdown output.
- GTD Workspace for Inbox, current work, this week, later, Waiting For, Someday, and Review.
- Settings interface in Dutch and English.
- Configurable recurrence phrases mapped to English Tasks instructions.
- Singular and plural recurrence intervals, including multi-week and multi-month patterns.
- Up to two people per task, both visible in the Workspace.
- Focus positions 1, 2, and 3 without focus hashtags.
- Responsive Workspace and settings layouts for desktop, iPad, and phone.
- Weekly and monthly review templates.
- Updated English and Dutch documentation.
