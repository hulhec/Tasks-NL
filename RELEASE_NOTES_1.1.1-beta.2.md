# Tasks NL 1.1.1-beta.2

## Changed

- Removed the visible `Tasks NL/Settings.md` synchronisation file.
- Tasks NL now exclusively uses Obsidian's standard plugin `data.json`.
- Removed creation and watching of the visible `Tasks NL` settings folder.
- Added NL/ENG synchronisation guidance directly below the language selector.
- Added a link to the official Obsidian Sync settings guide.
- Expanded the main README and both manuals with synchronisation instructions.
- Existing user settings and custom templates remain untouched.

## Obsidian Sync

Enable community plugins and plugin settings under **Settings → Sync → Vault configuration sync** on every device. After the first sync, fully quit and restart Obsidian.
