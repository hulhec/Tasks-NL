# Changelog

## 0.9.1 – Zelfstandige terugkerende taken

- Tasks NL verwerkt terugkerende taken nu zelf bij afvinken.
- `🏁 delete` vervangt de afgeronde taakgroep door de volgende herhaling.
- `🏁 keep` bewaart de afgeronde taak, voegt een voltooiingsdatum toe en maakt de volgende herhaling.
- Subtaken worden meegenomen en opnieuw opengezet.
- Dubbele verwerking van Obsidian-bestandsevents wordt voorkomen.
- Voor deze herhaalfuncties is de community Tasks-plugin niet meer vereist.
