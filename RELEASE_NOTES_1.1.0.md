# Tasks NL 1.1.0

## Changed
- Centralised all plugin settings in Obsidian's standard `data.json`.
- Removed the secondary vault settings mirror.
- Added safe migration that only supplies missing fields.
- Existing projects, people, repeat definitions and customised templates remain unchanged.
- New installations receive only minimal neutral examples.
- Preserved the desktop Workspace layout.
- Improved mobile Workspace readability by placing metadata on additional lines where needed.

## Sync
When using Obsidian Sync, enable synchronisation of settings and community plugins so `data.json` is available on all devices.
